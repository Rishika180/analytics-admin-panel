# Analytics Admin Panel

A responsive admin dashboard built using React.js and Material UI.
This project focuses on frontend architecture, reusable components,
dashboard UI patterns, and data visualization.

## Features
- Sidebar and topbar navigation
- Dashboard cards and charts
- Light / Dark mode
- Responsive layout
- Modular component structure

## Tech Stack
- React.js
- Material UI
- JavaScript

## Project Status
Frontend-focused project using mock data. Backend-ready architecture.
